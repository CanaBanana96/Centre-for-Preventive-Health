IMPORTANT STEPS:
==============

1. CREATE FOLDERS:
   - Right-click in your WordPress-CPH folder
   - Create "assets" folder
   - Inside assets, create "css", "js", and "images" folders

2. ADD YOUR IMAGES:
   - Add your actual logo files
   - Add doctor's photo
   - Add consultation/clinic images
   - Use optimized web formats (JPG for photos, PNG for logos)

3. UPDATE CONTACT INFO:
   - Replace all "+91-XXXXXXXXXX" with your phone number
   - Replace "info@centreforpreventivehealth.com" with your email
   - Update WhatsApp links with your number

4. TEST LOCALLY:
   - Open index.html in your browser
   - Check all links work
   - Test contact forms
   - Verify mobile responsiveness

5. DEPLOY:
   - Upload all files to your web hosting
   - Or use services like Netlify, Vercel, or GitHub Pages

File Sizes Recommendations:
==========================
- Logo files: Under 50KB
- Doctor photo: Under 200KB
- Other images: Under 500KB each
- Optimize images before uploading
